What it is

A C++20 real-time analytics engine that ingests NBA play-by-play (live + historical), computes statistics in real time, and serves them over HTTP and WebSocket. The point of the project is the systems engineering: everything below the analytics — the HTTP server, the WebSocket framing, the lock-free queue, the vector index, the cluster coordination — is written from scratch, not imported.
